#Пробная работа для знакомства с React и синтаксисом JSX
 Простой SPA.
 
 
## Инструкция:
- Склонировать репозиторий: git clone https://github.com/marityz/work-with-react.git

Для установки:
- npm install

Для разработки можно запустить сервер на http://localhost:3000:
- npm run start

Для сборки проекта:
- npm run build
